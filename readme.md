# MDL2OBJ
A tool made for [GodotVMF](https://github.com/h2xdev/GodotVMF). This tool based on [source-mdl](https://github.com/lordvonadel/source-mdl) by [LordVonAdel](https://github.com/lordvonadel)

## Dependencies
- Node v18+

## Build
```
npm install

npm run build

# macOS
npm run build:macos

# linux
npm run build:linux
```
